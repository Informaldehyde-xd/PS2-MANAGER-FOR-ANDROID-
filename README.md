This is the OPL manager alternative for android devices

FEATURES -------
1.Rename or edit your existing iso as well as ul games 
2.Fetch the art works simply for your iso or ul game
3.If no match found for your game you can select the desired art to that game
4.Works both on local storage or USB Drives
5.ul.cfg Generation for adding or scanning new game titles

NOTE -------
1.It requires Internet Access for fetching artwork and title 
2.If issue occurs you can free post it here
